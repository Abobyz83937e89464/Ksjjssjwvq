# Ksjjssjwvq
Базовый статический сайт для платформы morpheusov.com.

## Файловая структура
- `index.html` — главная страница.
- `style.css` — файл стилей (закругленные кнопки, адаптивный дизайн).
- `app.js` — клиентский скрипт для функциональности (био, галерея).
- `README.md` — этот файл.

## Функционал (базовый)
1.  Переключение на раздел "Био" по кнопке.
2.  Редактирование и сохранение био-описания (в LocalStorage браузера).
3.  Добавление изображений в галерею по URL.

## Развертывание на GitHub Pages
1.  Загрузите все 4 файла в корень репозитория.
2.  В настройках репозитория (`Settings` -> `Pages`) выберите источник `Deploy from a branch`, ветку `main` и папку `/ (root)`.
3.  Сайт будет доступен по адресу: `https://[ваш-логин].github.io/morpheusov-site/`

## Домен
Планируется привязка кастомного домена `morpheusov.com`.
